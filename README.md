# SolSplitter

**Instant split-screen launcher for Solana Seeker**

Open any two apps side-by-side with one tap. Save your favorite pairs and pin them to your home screen.

### Features
- One-tap split screen for any two apps
- Save and manage favorite pairs (Wallet + DEX, Charts + Trading, etc.)
- Pin saved pairs directly to home screen
- Clear visual selection feedback
- Beautiful Solana-themed dark UI

### Installation
1. Download the latest `app-release.apk` from [Releases](https://github.com/trademark979/SolSplit/releases)
2. Transfer to your Solana Seeker
3. Install and enjoy split-screen multitasking

### Pro Tip
For best results on first use, open both apps manually once before using split launch.
